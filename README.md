# ZERP-MD
Genesis port of my MS-DOS game

## Online emulator page

**Playable link:** https://fair-percentage.github.io/zerp-md/

**Controls:**
- Arrows - D-PAD
- Z, X, C - A, B, C
- Enter - START
- G - MODE

## Tools and programs used
- Genesis emulator: https://github.com/lrusso/Genesis
- SGDK: https://github.com/Stephane-D/sgdk
